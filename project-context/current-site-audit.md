# Current Site Audit: Ives Torres Foundation

## 1. Overview

Ives Torres Foundation is a nonprofit organization focused on supporting families and people with intellectual and developmental disabilities across California, with a strong focus on Hispanic/Latino families.

The foundation provides education, resources, events, community support, advocacy tools, and guidance for families navigating systems such as special education, Regional Centers, health services, self-advocacy, and inclusion programs.

The current website is live at:

https://www.ivestorresfoundation.org/

The current site appears to be built with Square / Weebly / EditMySite. It includes institutional content, donation functionality, events, resources, news, contact forms, regional pages, and bilingual content.

The website has a strong mission and valuable content, but the current experience feels template-based, visually dated, and somewhat disorganized. The main problem is not the lack of content. The main problem is how the content is structured, presented, and experienced by users.

The new website should preserve the foundation’s mission, content, and community focus, but rebuild the experience from scratch with a cleaner, warmer, more professional, modern, bilingual, and mobile-first design.

The new version should feel trustworthy, human, joyful, accessible, and easy to use.
