# Design Direction

## Visual Goal

The website should feel:

- Modern
- Warm
- Human
- Professional
- Trustworthy
- Clean
- Family-centered
- Accessible
- Joyful
- Community-driven

It should not feel like a store, template, corporate site, or government website.

## Emotional Direction

The website should make visitors feel:

- Welcomed
- Supported
- Safe
- Hopeful
- Connected
- Confident

Families visiting the site should quickly understand that the foundation is there to help them.

## Style Keywords

- Warm nonprofit
- Modern community organization
- Clean cards
- Soft shadows
- Rounded corners
- Large photography
- Gentle motion
- Strong accessibility
- Mobile-first
- Clear CTAs

## Color Direction

Use the current brand as inspiration but make it more refined.

Suggested palette:
- Deep blue for trust and structure
- Warm yellow/gold as accent
- White and soft off-white backgrounds
- Light blue sections
- Dark navy text
- Green or teal accents only if needed

Avoid:
- Too many bright colors
- Low contrast text
- Heavy gradients
- Overly childish colors

## Typography

Use clean, modern, accessible fonts.

Suggested:
- Headings: a friendly modern sans-serif
- Body: highly readable sans-serif

The typography should feel professional but warm.

## Layout

Use:
- Spacious sections
- Clear visual hierarchy
- Large hero section
- Cards for programs/events/resources
- Strong section headings
- Alternating light backgrounds
- Mobile-first layout
- Sticky or clean header

## Animation

Use subtle animations:
- Fade in on scroll
- Slide up cards
- Gentle hover effects
- Smooth page transitions
- Button microinteractions

Avoid:
- Excessive animation
- Distracting effects
- Heavy parallax
- Motion that hurts accessibility

Respect reduced motion preferences.

## Components

Main components:
- Header
- Footer
- Hero
- CTA buttons
- Program cards
- Event cards
- News cards
- Resource cards
- Impact stats
- Testimonial cards
- Facebook highlight cards
- Donation section
- Contact form

## Accessibility

The site must be accessible:
- Good color contrast
- Clear focus states
- Large readable text
- Semantic HTML
- Keyboard-friendly navigation
- Alt text support for images
- Forms with labels
