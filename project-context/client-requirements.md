# Client Requirements

## Client

Ives Torres Foundation

## Main Request

The client wants a new version of the current website. The current website is live and built with Square/Weebly, but the goal is to build a cleaner, more modern, more professional, and more intuitive website.

The new site should have a strong visual impact and feel trustworthy, warm, and easy to use.

## Main Goals

- Create a modern and professional website
- Improve UX and UI
- Make the site easier to navigate
- Make the foundation feel more trustworthy and established
- Improve donation experience
- Improve events experience
- Make resources easier to find
- Support Spanish and English
- Keep costs as low as possible
- Keep the existing domain
- Keep email working

## Known Infrastructure

- Current website built with Square/Weebly
- Domain managed in Porkbun
- Email currently managed with Hostinger/Titan
- Donations can use PayPal and Zelle
- Facebook is an important source of photos/videos/events

## Client Preferences

- Keep costs low
- Use secure services
- Avoid unnecessary monthly costs
- Use PayPal and Zelle for donations
- Include a curated Facebook section
- Allow the foundation to update events/news/resources if possible

## Important Notes

- Do not break existing email.
- Do not touch DNS records related to email unless required.
- Avoid making events look like products.
- Avoid making the site feel like a store.
- The website should be easy for families to use.

## Content Accuracy Rules

All public-facing copy must be based on the existing Ives Torres Foundation website and client-provided context.

Do not invent:
- Founder biographies
- Staff roles
- Personal details
- Impact metrics
- Program claims
- Donation outcomes
- Legal/tax claims
- Event details

Important:
Ives Torres is male and has Down syndrome. He is the inspiration behind the foundation. Do not describe Ives as a female leader, staff member, founder, or executive.

If exact information is not available, use neutral foundation-level copy instead of inventing details.
